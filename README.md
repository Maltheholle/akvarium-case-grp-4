#Akvarium Interface

Et interaktivt akvarium lavet i HTML, CSS og JavaScript.

##Sådan bruger du det

- Klik på en fisk for at få information
- Der kommer en tekstboks frem med info om fisken
- Fisken afspiller også en lyd, så den “snakker”
- Klik på skattekisten for at gå videre til vendespillet. Vendespillet har sin egen README fil. 

##Funktioner

- Interaktivt akvarium
- Klikbare fisk med information
- Tekstboks med fakta om fiskene
- Lyd når fisken vælges
- Skattekiste der fører til vendespillet

##Teknologier

- HTML5
- CSS3
- JavaScript (DOMmanipulation og interaktivitet)
